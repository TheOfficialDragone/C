# C Exercises

A collection of **C programming exercises** covering a variety of topics and real-world scenarios.

## Projects

| Folder | Description |
|--------|-------------|
| `Assenze` | Absence tracking system |
| `Cambio_valuta` | Currency exchange calculator |
| `Codice_Morse` | Morse code encoder/decoder |
| `Durata_CD` | CD track duration calculator |
| `Quesiti` | Miscellaneous exercises and problems |
| `Serie_televisive` | TV series management |
| `Unisci_file` | File merging utility |
| `alberghi` | Hotel management system |
| `converti_Binario` | Binary number converter |
| `ricerca_nomi` | Name search utility |

## Topics Covered

- File I/O
- Arrays, strings, and pointers
- Structs and dynamic memory
- Algorithms and sorting
- Data conversion

## Requirements

- GCC or any C99-compatible compiler

## Compile & Run

```bash
gcc -o program file.c
./program
```
